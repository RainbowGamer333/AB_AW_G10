Règles

  • Le but du jeu est de découvrir toutes les cases libres sans faire exploser les mines.
  • Les chiffres indiquent le nombre de mines autour de la case.
  • Un clic droit permet de placer un drapeau sur une case.
  • Un clic gauche permet de découvrir une case.
  • Le jeu est perdu si une mine est découverte.
  • Le jeu est gagné si toutes les cases libres sont découvertes.
